# 🐝 Make Bee Trip – Travel Booking Landing Page

A modern and responsive travel booking landing page built using **React JS** and **Node.js**, showcasing flight, train, bus, and hotel booking features with API integration.

---

## 🚀 Features

* ✈️ Flight Booking (API-based search)
* 🚆 Train Booking (API-based search)
* 🚌 Bus Booking (API-based search)
* 🏨 Hotel Booking (Manual + API-based)
* 🔐 Login / Signup Modal (UI)
* 🌐 Fully Responsive Design
* ⚡ Clean UI with modern layout

---

## 🛠️ Tech Stack

### Frontend:

* React JS
* Tailwind CSS
* Axios / Fetch API

### Backend:

* Node.js
* Express.js

## 🔗 API Endpoints

| Method | Endpoint     | Description    |
| ------ | ------------ | -------------- |
| GET    | /api/flights | Search flights |
| GET    | /api/trains  | Search trains  |
| GET    | /api/buses   | Search buses   |
| GET    | /api/hotels  | Get hotels     |

## ⚙️ Setup Instructions

### 1. Clone Repository

git clone https://github.com/Muskan753951/make-bee-trip.git

---

### 2. Install Dependencies

#### Frontend:

cd client
npm install

#### Backend:

cd server
npm install

---

### 3. Run Project

#### Backend:

npm start

#### Frontend:

npm run dev
